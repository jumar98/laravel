Clases de PHP

	Mundo Web

1 - Cliente pide la pag. web en el navegador

2 - Llega la peticion del sitio web al servidor web

3 - Servidor procesa la informacion

4 - Servidor regresa HTML

5 - www envia HTML al cliente

6 - Navegador interpreta el HTML 

	Servidores 

- PC que ofrece servicios y recibe peticiones

- Compartidos: Rentan espacio pequeño, en una carpeta. 
  (Riesgoso no escalable)

- S. Virtuales VPS: Virtualizan y parte del hardware se
  son dedicados.

- S. Dedicados: Maquinas al 100% para nosotros (Mas costosos)
  
- Computo en la nube: Muchos servicios a solicitud.

	Back End

- Del lado del servidor son procesados interpretan las 
  peticiones y lo regresan en html, css y js
  (Conocer node, python)

	Front End

- Conjunto de html, css y js. Recoger las entradas del
  cliente y muestra resultados. Interpretado en el
  navegador.

	Conexion al Servidor

- FTP: Protocolo de comunicacion de archivos entre dos
  computadoras (Fizilla)

- Git: Sistema de control de versiones

- GitHut: Servicio que utiliza git, es un servidor, 
  red social.

	Bases de Datos

- Software para almacenar la informacion 

	Servidor web

- Programa que recibe las solicitudes 

- Apache: Esta diseñado para ser dinamico

- NGINX: Es lento para procesar paginas dinamicas,
  solo para dar informacion estatica.

	Sistemas operativos

- PHP es multiplataforma

	Entorno de desarrollo

- Espacio donde instalamos programas para que funcione
  nuestro aplicativo.
